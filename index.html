1. Reunión informativa

Se realiza una evaluacion de las necesidades del proyecto

2. Evaluación

Se propondra el mejor enfoque según los requierimientos del proyecto aplicando diseño centrado en el usuario.

3. Diseño

Aquí es donde ocurre la magia. Trabajaré en Concepto y Diseño. Luego se presentará el prototipo

4. Iteracion

Se realizarán pruebas de usabilidad para iterar el diseño logrando el mejor producto final.

5. Implementación

Desarrollo del código y despliegue en la web.

6. Mantenimiento

Posibilidad de gestionar el proyeco a largo plazo.
